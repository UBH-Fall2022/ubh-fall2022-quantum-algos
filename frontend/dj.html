Deutch-Josza

The Deutch-Josza algorithm is a quantum algorithm that solves the problem of determining whether a function is constant or balanced. This is a problem that can be solved in polynomial time with a classical computer, but the Deutch-Josza algorithm is the first quantum algorithm that solves a problem in polynomial time.
