Grover's Algorithm

Grover's algorithm is a quantum algorithm that finds an element in a list of unstructured items in O(sqrt(N)) steps, where N is the number of items in the list. This is a significant improvement over the best classical algorithm, which would require O(N) steps. The algorithm was invented by Lov Grover in 1996.
